Description
===========
Dead simple experimental module that checks a website inbound links status.
Efficient module to restrict your customer from root of his/her website,
The purpose of this module is constraint your customer for on time settelment, 
so by enable this module, your customer can access his/her site only form
http://www.customerdomain.com/index.php
but not from http://www.customerdomain.com/ 

Dependencies
============
  - nothing
